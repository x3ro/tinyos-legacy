Windows Download and Installation

TinyOS Release 0.1.5
Installation setup.exe Version 1.1
January 12, 2002

TinyOS is installed using Installshield. Download the TinyOS setup via anonymous ftp from ftp://mmnt.millennium.berkeley.edu/setup.exe (approximately 165MB) or request a CD from the TinyOS group.

Place the Installshield CD in the drive or, if downloading from a web page, click on the setup.exe link.The following table describes the windows that come up in order and what action should be taken on each window. Some windows take a few seconds to come up, so please be patient.

AVRGCC - Setup error

The text says “Access is denied - outreset”. This window is alarming, but normal. Click OK.

AVRGCC - Installation Complete

Click OK. Another DOS window will come up in the background and you’ll see many commands scroll by. The DOS window will likely continue to remain for a minute. This is normal. Do not do anything to that window.

Information - Removed old c:\uisp.tar

OK. (This window may not come up if you don’t have old tar files on your disk.) A DOS window will come up in the background and you will see a list of files. Don’t do anything.

C:\cygwin\bin\bash.exe

This DOS window asks if you would like to continue installing the GiveIO driver. Type “Y” and <RETURN>. A few more compilation instructions display and then the window will disappear.
