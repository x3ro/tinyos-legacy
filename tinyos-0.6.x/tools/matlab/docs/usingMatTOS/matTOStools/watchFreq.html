watchFreq

Watch freq plots a histogram of the data arriving from the serial port.  This is useful for seeing the frequency with which different values are arriving.  For example, the default field that is being watched is 'moteID', so you can see the frequency with which you are hearing each mote.  You might also want to use it to characterize the noise in your signal or to find a probability distribution over your signal.   It is otherwise exactly the same as the other two watch tools.
