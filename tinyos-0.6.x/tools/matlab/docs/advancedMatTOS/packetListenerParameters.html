Passing Parameters to Packet Listeners

When registering a function of yours as a packet listener, you can send a parameter to the function each time, along with the packet, in the following way:

addPacketListener(packetPort, {'myfunction', x})

In this case, x is some parameter that you want to pass. 

X will not change at all ever because it is always passed by reference.  However, if you make it an object, you could make it a wrapper around a database of persistent global variables.  To see how to do this, see how I implemented the packetPort class.  Doing this would allow you to send a different parameter to a function each time a packet is received from a different mote.
