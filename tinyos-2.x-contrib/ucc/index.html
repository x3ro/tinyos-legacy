About MISL

The MISL is a research laboratory within the Department of Computer Science at University College Cork. The Laboratory conducts research in the areas of computer networking and systems, with a focus on applications in the mobile/wireless and multimedia domains.

The Preemptive Priority Scheduler (PL Scheduler)

This contrib project provides preemptive task scheduling capabilities for TinyOS-2.x. 5 priority tasks queues are provided to facilitate programmers with more fine-grained control over their tasks.

This research is based on a publication "Adding Preemption to TinyOS", [download]
For more information, please visit my website http://www.cs.ucc.ie/~cd5.

Usage

define priority tasks using by wiring up interfaces, an example can be seen in the apps directory, compile with:
make telosb priority

Install

MakeSetup will copy all the files to your tinyos directory if you have the TOSROOT variable defined, and you must also have write permission.
To use type make -f MakeSetup install
