Class GenericTopo

source code

         object --+    
                  |    
Manager.NodeManager --+
                      |
                     GenericTopo

Creates a generic topology (that is not tied to TOSSIM).

This can be useful to generate and save different topologies that can be examined and/or loaded later without needing to integrate with TOSSIM.

Since these are not real motes, the only TOSSIM Mote-like property they have is an ID.
