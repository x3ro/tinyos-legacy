Mts4x0 alternative implementation

Driver for mts420 sensor board, mainly supports the leadtek GPS,

It has a generic support for the hardware switches (adg715) and is organized in a portable way, with chips/platforms and sensorboards directories.
It takes a careful path to make proper initialization and shut down of the switch devices on the board,
watch error status and try to keep everything in a coherent state even when errors occurs.

Feel free to mail me for more information, bug report, patches etc...
