README for TestGpsParser
Author/Contact: Aurélien.francillon@ inria.fr

Description:

Sample application to test the on mote parsing of gps frames the
converted data is sent over the serial line in GpsGGAMsg packets this
data can be displayed with pc/testGPS.py program

Tools:
pc/testGPS.py

Known bugs/limitations:

- VTG Frames not handled 
- pc/testGPS.py serial port to use is hardcoded
