ECH2O-TE Telos Interface

Impact Lab
Arizona State University

Overview

This contribution is an interface to a Decagon Devices ECH2O-TE soil probe from a TelosB class mote. The contribution includes two parts -- hardware and software.

  • The hardware contribution is a detailed description of the circuitry we used to interface the TE stereo plug with the TelosB's UART and ADC pins.
  • The software portion is the TinyOS module to drive the hardware interface. See the module and configuration comments for details.
The end result is a Read interface that provides the TE's dielectric (moisture), temperature, and conductivity readings.

As of Sep 2008 the prototype has a 95% reading success rate since installation in Dec 2007. The results are informative as you can see from the latest graphs.
