Cryptography algorithms for TinyOS

The aim of this project is to provide Cryptography algorithms in TinyOS 2.x. All contributions are welcomed.

Implementation done:

Block cipher

AES

We provide an interface which provide AES block encryption and decryption. This implementation is based on the Public Domain implementation of Karl Malbrain. For more information about this implementation, consult the source code documentation.

Stream cipher

Trivium

The Trivium stream cipher is now implemented as a TinyOS interface for 8-bit and 16-bit microcontroller. This implementation follows the original implementation available in the eSTREAM repository. Some applications using this interface are available in the apps folder of the project. For more information consult the cipher webpage.
For more information about this implementation, consult the source code documentation.

Universal hash functions

MMH

We have implemented the MMH universal function as a TinyOS interface. This function was proposed in the paper of Shai Halevi and Hugo Krawczyk : MMH: Software Message Authentication in the Gbit/Second Rates. We also provide an application which uses this interface and the Trivium interface to construct a Wegman-Carter authentication. This application is available in the apps folder.

Poly32

We have implemented the Poly32 universal function as a TinyOS interface. This implementation follows the original paper of Ted Krovetz and Phillip Rogaway : Fast Universal Hashing with Small Keys and No Preprocessing: The PolyR Construction . We also provide an application which uses this interface and the Trivium interface to construct a Wegman-Carter authentication. This application is available in the apps folder.

For more information about thesw two implementations, consult the source code documentation.

Download

The CVS directory can be browsed at the address.
The directory can be downloaded with the CVS command:
cvs -z3 -d:pserver:anonymous@tinyos.cvs.sourceforge.net:/cvsroot/tinyos checkout -P tinyos-2.x-contrib/crypto/

Installation

To install or remove all the interfaces provided by this directory in the TinyOS tree, the scripts install.sh and uninstall.sh can be used. They must have the execution permission with the command:
chmod +x install.sh uninstall.ch
Then the command:
./install.sh
copies all the files in the TinyOS tree.

For each application example in the folder apps, there is a Makefile which can compile the application the same way as standard TinyOS applications.
