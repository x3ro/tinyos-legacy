Interface: tos.interfaces.Poly32

interface Poly32

Interface for the Poly32 universal hash function family in TinyOS. This Interface must not be used alone. It needs a block cipher or a stream cipher to construct a Wegman-Carter authenticatior.

Author:
Sylvain Pelissier <sylvain.pelissier@gmail.com>
