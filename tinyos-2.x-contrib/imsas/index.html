Projects for TinyOS tinyos-2.x-contrib



Project 1: Intelligent Container
A variety of apps and sensor support code will be added soon. Note that only the published and older code can be added as the project is ongoing and under research.


Project 2: Sht75 on Meshbean platform
The code pertaining to this is here. An example application is in the app folder. The folder reads sht11. However, you can rename as you wish, and the code itself can be used for both sht1x and sht7x series of Sensirion temperature/humity sensors. These codes have been ported from telosb to meshbean platform (Much thanks to Philipp Sommers et al.). Beginners may find this white paper useful (download and view with a pdf viewer).


Project 3: Nokia6100 LCD display on Telos/Meshbean platform
This display uses the SPI. Code is available for both Telos and Meshbean platforms. This is pretty useful for onboard debugging purposes without being restricted to 3 LEDs or having to send all data to a host pc for evaluation. Code will be added soon after bug testing.
