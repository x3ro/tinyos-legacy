README for Blink.INT4
Author/Contact: Laurynas.Riliskis@ltu.se

Description:

Blink is a simple application to test int erupt on pin INT4 and ad converter 
(adc) on the expansion board. 
Put jumper on INT4 and AN0. compile and install application. Run it. Then you
press interrupt button the green LED will turn on if the value of the adc is
higher then the threshold. The LED will be turned of then you decrease adc value
under the threshold.


Known bugs/limitations:

None.
