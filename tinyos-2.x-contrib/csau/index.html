TinyOS Contribution from Aarhus University

Below you will find a list of maintained projects from Aarhus University. For any questions or comments please contact: mth at cs dot au dot dk.

Unified Broadcast

Code for unifying broadcast in TinyOS. For further details see paper titled Unified Broadcast in Sensor Networks from IPSN 2011.

Misc

This project contains miscellaneous libraries that can be used in different projects

Generic Neighbor Table

misc/tos/lib/neighbortable

A generic neighbor table that can be extended by creating new components. See an example of how to extend it in neighbortable/example.

Time Measures

misc/tos/lib/timemeasures

A library for supporting various time measures in TinyOS. This can be used in protocol or implementation evaluations.

Quick empirical time measured can be done by using the Quicktime components which can be controlled by calls to time_start(), time_stop(), and time_get().

Random Back-off Timer

misc/tos/lib/timer

A TimerTMilliC with a random back-off. The timer keeps its period, but always fires with [period/2, period]. This can be used to avoid timers on different motes firing at the same time.
