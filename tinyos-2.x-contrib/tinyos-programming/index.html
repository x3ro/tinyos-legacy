TinyOS Programming

This directory contains the example applications from the "TinyOS Programming" book, published by Cambridge University Press, ISBN 978-0521896061. The applications are developed for micaz motes with an mts310 sensor board, most of them should be easily ported to other motes and/or sensor boards.

You can get the source code from TinyOS's CVS repository on sourceforge.net using the following two commands:

    cvs -d:pserver:anonymous@tinyos.cvs.sourceforge.net:/cvsroot/tinyos login
    cvs -z3 -d:pserver:anonymous@tinyos.cvs.sourceforge.net:/cvsroot/tinyos co -P tinyos-2.x-contrib/tinyos-programming

Or using the convenient source tarballs linked below.

  • AntiTheftBook: the successive "anti-theft" applications developed in Chapter 6.1-6.5. [source]
  • FlashSampler: the "sample-to-flash" application developed in Chapter 6.5.3. [source]
  • ReliableSerial: the simple reliable transmission protocol developed in Chapter 7.5. [source]
  • SoundLocalizer : the "closest-to-sound" application developed in Chapter 13. [source]
