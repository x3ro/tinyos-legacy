AVR toolchain

This subproject contains AVR toolchain for OpenSolaris followers. The package sources are scripted for pkgbuild (rpmbuils/.srpm/.spec like) building environment. Some prerequisity sofware is needed from SFE. The software is installed "/usr/gnu/" prefix (some configs "/etc/gnu/").

  • SPEC definitions - The *.spec package definition files to build AVR toolchain that is compatible with TinyOS. The reffered patches are under patch subdirectory. I use it for my other TinyOS projects.
  • precompiled packages - Precompiled SVR4 packages (pkgs stream format) for x86 platform. (I use IPS packages from pkgbuild.)

Versions of tools:

  • AVaRICE - 2.12 (previous 2.10)
  • GNU binutils for AVR - 2.19 (previous 2.17)
  • GNU gcc for AVR (only c language) - 4.5.3 (previous 4.1.2)
  • GNU gdb for AVR - 7.3
  • AVR libc - 1.7.1
  • AVRDUDE - 5.10
  • nesC - 1.3.3

I use TinyOS directly from SVN repository. How to build TinyOS from SVN see wiki.
