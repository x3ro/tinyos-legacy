Cooperative TinyOS Scheduler

This contrib project provides Cooperative task scheduling capabilities for TinyOS-2.x. The Cooperative Scheduler for TinyOS 2.x supports new Job interface. Jobs are suitable for less critical and long computations because they are cooperatively preemptable. In this Scheduler basic TinyOS tasks have higher priority than jobs.

The main difference between Job and TinyOS regular task is that Job allows multiple entry points and suspending and resuming of execution at certain locations of its code but TinyOS task has one entry point and execute in run-to-completion manner.

Installation & Usage :

Copy entire project tos directory to tos directory of your TinyOs Installation or place all project files into your application directory. Then go to shell and type:
$make mica2
$make micaz
Currently supported platforms are mica2 & micaz (ATMega128L based platforms). There are two sample application exists in apps directory.
