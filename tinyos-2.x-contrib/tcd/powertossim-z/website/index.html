PowerTOSSIM is a power modelling extension to TOSSIM created in 2004 for TinyOS, version 1.x. The design models power consumption for the mica2 sensor mote. With the release of TinyOS 2.x (T2), TOSSIM was also redesigned to compile for the micaZ mote architecture. PowerTOSSIM was also updated by the authors in the paper entitled Simulation blocks for TOSSIM-T2 to target the new T2 TOSSIM implementation but only for the mica2 mote.

PowerTOSSIM-Z was designed with a two fold approach:
  1. Fully port PowerTOSSIM for T2, extending some of the important components for micaZ (ATM128 Microcontroller, AT45DB Flash, Radio Stack Chipcon CC2420).
  2. Extend the simulator to model non-linear battery effects (Rate Capacity Effect, Recovery Capacity Effect).
The implementation modifies some chips and introduces debug statements at strategic points in TOSSIM's code. The output file containing the debug statements is processed to obtain the power consumption per node at each time. This takes into account the energy model of the micaZ for different components and the non-linear battery effects.

Finally, the postprocessor can work at runtime when TOSSIM is being executed. PowerCurses has been implemented as a tool to graphically show the power consumption per node at runtime when in verbose mode.
