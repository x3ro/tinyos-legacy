About dexma

dexma sensors sl is a company located in Barcelona specialized in software for wireless sensor networks. Our main product, dexcell (still in development) aims to provide and easy way to create and deploy vertical solutions based in wireless sensor networks. dexCell offers a SOA lowpower based stack and a integration and information management server.

You can get more information about dexma and its products and services, in our website: www.dexmasensors.com.

What's inside this repository

While dexCell is still being developped, as a TinyOS developers we have created some small applications and libraries that could be usefull to other developers.

For the moment we offer two small projects.

dexCell skeleton generator

Using this perl script you can easy generate a skeleton for a TinyOS based application. Just follow the indications on the screen. There are some bugs when using dissemination. For the moment works fully with telosb / tmote platforms.

Radio printf

Based in the printf library (that prints a string over the serial port) this modification sends this string throught a broadcast radio message. Just plug a mote with a BaseStation installed and the Java tools provided by the printf library.
