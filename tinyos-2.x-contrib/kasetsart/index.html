IWING's TinyOS Projects

This page briefly describes two TinyOS-related projects being developed by the Intelligent Wireless Network Group (IWING) at Department of Computer Engineering, Kasetsart University. The two projects are: (1) TinyOS Port for IWING-MRF sensor mote hardware, and (2) TosVis, a visualization tool for TOSSIM.

IWING-MRF Platform

This project is to implement TinyOS on the IWING-MRF platform. IWING-MRF is a low-cost, AVR-based mote hardware. Main features include
  • ATMega328P microcontroller clocked at 12MHz, 32KB flash, 2KB SRAM.
  • MRF24J40MA, IEEE802.15.4 transceiver module from Microchip.
  • Reprogrammable via USB port using the built-in bootloader based on USBaspLoader. No extra usb-to-serial adapter or chip is required.
  • Easy-to-make single-sided PCB.
Run the install.sh script to let your current TinyOS installation know the IWING-MRF platform. 
sudo -E ./install.sh
You can then build and install your application on an IWING-MRF mote by running: 
make iwing-mrf
make iwing-mrf install

TinyOS Visualization

TinyOS Visualization (TosVis) is a graphical visualization tool for TinyOS Simulator (TOSSIM) It is written in Python and based on the TopoVis library, another tool developed by IWING to create snapshots and animations for wireless ad hoc and sensor networks. TosVis collects and analyzes debugging messages produced by TOSSIM such as LEDs and radio activities. More activities can be easily visualized by extending the provided Python base class.
