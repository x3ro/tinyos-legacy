Operand Stack Heap Condition Codes
  • Initial (operands):
    variable 1
    variable 2
  • Final (return values):
    variable 1 - variable 2

The heap is read if a variable is a location to determine which axis to multiply.

  • 1 = x
  • 2 = y
  • both otherwise

N/A

Description

Pop two variables off the stack and push their difference onto the stack.

If adding a location and value, heap[15] is used: if 0 perform subtraction on both x and y, if 1 only subtract from x, if 2 only subtract from y.
