Operand Stack Heap Condition Codes
  • Initial (operands):
    N/A
  • Final (return values):
    Value: the condition code
N/A Accesses the condition code, but does not modify it.

Description

Pushes the agent's condition code onto its stack as a value. The condition code is 16-bits long and is used to indicate whether certain operations (e.g., migration) completed successfully, or whether to jump.
