wclone

Class: B
(legend)
Operand Stack Heap Condition Codes
  • Initial (operands):
    location or value
  • Final (return values):
    N/A
N/A
  • 0 - clone failed (agent continues to run on original host)
  • 1 - clone succeeded, this is the clone
  • 2 - clone succeeded, this is the parent
  • 3 - clone failed due to lack of memory at destination, this is the parent

Description

Clones an agent to a remote sensor. The remote sensor is specified by the location or value on top of the stack. Since this is the weak version, the clone resumes running from the beginning.
