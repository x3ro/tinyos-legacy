Operand Stack Heap Condition Codes
  • Initial (operands):
    Template
  • Final (return values):
    Tuple if match is found, nothing otherwise
N/A

0 = operation failed (to matching tuple was found)

1 = operation succeeded (matching tuple found and pushed onto the operand stack)

Description

Searches the host's local tuple space for a tuple matching a template. If a match is found, remove it from the tuple space, push it onto the operand stack and set the condition to 1. If no matching tuple is found, set the condition to 0.
