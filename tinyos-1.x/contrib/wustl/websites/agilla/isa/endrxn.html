Description

Signals the end of a reaction. When a reaction fires, the code associated with the reaction is executed to completion without preemption. The last instruction of the reaction must be an endrxn to tell the Agilla VM that the agent may resume running regular code. endrxn updates the program counter to be its original value before the reaction fired.
