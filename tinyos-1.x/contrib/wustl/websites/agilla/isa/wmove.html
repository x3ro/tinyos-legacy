0 = move failed (agent is still on original host)

1 = move succeeded (agent is now running on remote host)

3 - move failed due to lack of memory at destination

Description

Performs a weak migration to a remote host. The location of the remote host is specified by the parameter on top of the operand stack. A weak migration does not save agent execution state. The agent restarts from the beginning when it arrives.

  • 0 - move failed (agent continues to run on original host)
  • 1 - move succeeded (agent now runing on remote host)
