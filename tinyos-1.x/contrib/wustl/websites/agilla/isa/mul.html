The heap is read if a variable is a location to determine which axis to multiply.

  • 1 = x
  • 2 = y
  • both otherwise

N/A

Description

Pops two variables off the stack and performs a multiplication. Valid parameters are:
  • Two values
  • Two locations
  • A a value and a location
  • A value and a sensor reading

If heap[11]=1, only consider the x-coordinate, else if heap[11]=2, only consider the y coordinate. Otherwise, consider both coordinates.
