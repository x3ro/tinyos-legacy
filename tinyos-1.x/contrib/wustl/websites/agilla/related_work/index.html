Mobile Agents in WSNs

  • Location Tracking in a Wireless Sensor Network by Mobile Agents and its Data Fusion Strategies, by Yu-Chee Tseng, Sheng-Po Kuo, Hung-Wei Lee, Chi-Fu Huang. In Lecture Notes in Computer Science, Volume 2634, IPSN'03, Jan 2003, Pages 625 - 641. A journal version appeared in: The Computer Journal 2004 47(4):448-460; doi:10.1093/comjnl/47.4.448

    Uses mobile agents to track sensed objects moving through a sensor field. Uses "master" agents that spawn "slave" agents to help with the tracking. Basic algorithm assumes a triangular network topology. Uses safe-distance-like technique to adapt protocol to work in irregular network topologies.
  • Distributed computing paradigms for collaborative signal and information processing in sensor networks.Xu, Y. and Qi, H. 2004. In Journal of Parallel Distributed Computers. 64, 8 (Aug. 2004), 945-959. DOI= http://dx.doi.org/10.1016/j.jpdc.2004.04.002

    Studies the performance difference between client/server and mobile agent systems.
  • Multisensor Data Fusion in Distributed Sensor Networks Using Mobile Agents, by Hairong Qi, Xiaoling Wang, S. Sitharama Iyengar, and Krishnendu Chakrabarty. In 4th International Conference Information Fusion (FUSION 2001), Montreal, Canada, August 7-10, 2001.
  • Multi-resolution data integration using mobile agents in distributed sensor networks, by Hairong Qi, S. Sitharama Iyengar, Krishnendu Chakrabarty, in IEEE Transactions on Systems, Man, and Cybernetics Part C: Applications and Reviews, vol. 31, no. 3, pp383-391, August, 2001.
  • High Performance Sensor Integration in Distributed Sensor Networks Using Mobile Agents, by Hairong Qi, Xiaoling Wang, S Sitharama Iyengar, and Krishnendu Chakrabarty. International Journal of High Performance Computing Applications August 1, 2002 16: 325-335.

  • Distributed Multi-Resolution Data Integration Using Mobile Agents, by Hairong Qi, S. Sitharama Iyengar, Krishnendu Chakrabarty. In Proceedings of IEEE Aerospace Conference, March 2001.
